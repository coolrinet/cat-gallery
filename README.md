# 🐈 Галерея котиков

Данное приложение представляет собой галерею с изображениями котиков. Реализовано в рамках тестового задания на вакансию стажера frontend-разработчика в Учу.ру

## Функционал

- Возможность добавлять котиков в категорию "Любимые" (они сохраняются на клиенте, поэтому при перезаходе они никуда не пропадут)
- Бесконечная прокрутка всех котиков
- Адаптивно под все устройства (ПК, планшет, смартфон)

## Основные технологии

- React 19
- Vite
- Typescript
- Ky - легкая обертка над fetch, аналог axios
- Tanstack Query - удобное получение данных для бесконечной прокрутки

## Инструкция по запуску локально

```
pnpm install
pnpm dev
```
